# ohlife-to-dayone
A Ruby Script to import OhLife text export into DayOne

Setup:

1. [Install DayOne CLI tool](https://dayoneapp.com/guides/tips-and-tutorials/command-line-interface-cli/)
2. Install Ruby via [Homebrew](https://brew.sh)

```sh
brew install ruby
```

Run the script
```sh
ruby transfer.rb
```
